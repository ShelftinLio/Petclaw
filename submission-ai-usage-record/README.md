# Petclaw AI 使用记录提交包

项目名称：Petclaw Desktop Pet
记录日期：2026-05-08
说明：此 Codex 项目与 `Petclaw_win` Codex 项目为同一参赛项目，本包按当前仓库 `Petclaw-git` 的设计文档、实现计划、提交历史与生成资源记录整理。

## 包内文件

- `01-Prompt设计.md`：展示如何把产品想法拆成可执行 Prompt，包括需求澄清、架构拆解、测试驱动、界面迭代、图片生成等。
- `02-迭代思路与开发日志.md`：按时间线记录从桌面 AI 伴侣到 Petclaw 成长系统、宠物工作室、Inbox 的迭代过程。
- `03-生成内容占比说明.md`：说明 AI 参与范围、人工参与范围和生成内容占比估算口径。
- `04-验证与人工把关.md`：记录测试、人工验收、安全边界与最终提交前的检查方式。
- `evidence/`：保留可追溯材料，包括 git 日志、AI 设计/计划文档副本、AI 图片生成 Prompt 与生成任务文件。

## 提交展示建议

提交作品时可直接上传本目录外层生成的 `Petclaw_AI使用记录_2026-05-08.zip`。如果平台允许补充说明，可以写：

> 本项目在 Codex 中采用“需求拆解 -> Prompt 设计 -> 设计文档 -> 实现计划 -> 测试验证 -> 人工验收”的 AI 协作流程。压缩包内包含 Prompt 设计、迭代日志、生成内容占比说明，以及设计/计划/图片生成等日志证据。

## 记录边界

本包没有包含 API Key、用户隐私数据、`.env` 配置、运行时本地缓存或个人账号信息。仓库中已有的设计文档和生成资源记录只复制必要文件作为提交证据。
