# Evidence 目录说明

本目录用于证明 AI 使用记录不是单独编写的说明，而是来自项目开发过程中的真实工作材料。

## 文件来源

- `superpowers-specs/`：功能设计文档副本，来自 `docs/superpowers/specs`。
- `superpowers-plans/`：实现计划文档副本，来自 `docs/superpowers/plans`。
- `imagegen/`：自定义宠物图片生成 Prompt、生成任务和生成资源副本，来自 `assets/pets/custom/custom-20260505-084436`。
- `git-log-recent.txt`：最近提交摘要，用于展示迭代节奏。
- `git-status-before-package.txt`：打包前工作区状态摘要。
- `git-diff-stat-before-package.txt`：打包前已有改动摘要。
- `package-file-inventory.txt`：压缩包目录文件清单。

## 使用方式

评审如果只需要快速了解流程，阅读根目录 4 份 Markdown 即可。若需要日志证据，可查看本目录中的设计文档、实现计划、git 日志和图片生成 Prompt。

## 隐私处理

本目录没有复制 `.env`、API Key、token、用户个人配置或运行缓存。图片生成记录仅保留 Prompt、任务说明、宠物 manifest 和生成结果，用于证明 AI 生成资源流程。
